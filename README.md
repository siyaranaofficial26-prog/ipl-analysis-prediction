# IPL Match Analysis & Win Prediction

## Overview
Analysed IPL match data from 2008-2024 using Python. Built 5 
visualizations and a Random Forest model to predict match winners.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Scikit-learn

## Key Insights
- Mumbai Indians and CSK have the most wins historically
- Teams strongly prefer fielding first after winning the toss
- Run scoring has increased significantly post 2021.

## Model
Random Forest Classifier — 50% accuracy on test data

## Files
- ipl_data_analysis.ipynb — main notebook
- matches.csv — match level data
- deliveries.csv — ball by ball data
