# IPL Match Analysis & Win Prediction

## Overview
Analysed IPL match data from 2008-2022 using Python. Built 5 
visualizations and a Random Forest model to predict match winners.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Scikit-learn

## Dataset
Download the dataset from Kaggle:
https://www.kaggle.com/datasets/patrickb1912/ipl-complete-dataset-20082020

Place matches.csv and deliveries.csv in the same folder as the notebook.

## Key Insights
- Mumbai Indians and RCB have the most wins historically
- Teams strongly prefer fielding first after winning the toss  
- Run scoring has increased significantly post 2022

## Model
Random Forest Classifier — 50% accuracy on test data
